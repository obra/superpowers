# /ace-knowledge-agent - Deprecated Alias of /ace-knowledge

`/ace-knowledge-agent` is deprecated. Please use `/ace-knowledge` instead.
This command remains for backward compatibility and should execute exactly
the same behavior as `/ace-knowledge`.

## Overview
Reuse all actions from `@.claude/commands/ace-knowledge.md` with the same `$ARGUMENTS`.

Example:
- Old: `/ace-knowledge-agent ingest docs/demo/em-book-intro.md`
- New: `/ace-knowledge ingest docs/demo/em-book-intro.md`
