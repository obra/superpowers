# Workspace Memory System

## Layers
1. Gigabrain — automatic recall + capture
2. PARA (`~/life`) — durable truth
3. `memory/YYYY-MM-DD.md` — daily residue
4. `MEMORY.md` — routing/index layer
5. OpenStinger — optional cross-session graph recall
6. lossless-claw (LCM) — session compaction + recovery

## Guardrails
- PARA is durable truth.
- LCM is not long-term truth.
- OpenStinger does not replace PARA.
- Bootstrap files stay small.
