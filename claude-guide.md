# Claude Code 기본 사용 가이드

## 처음 실행 시

```
> 1. Yes, I trust this folder  ← Enter 누르기
  2. No, exit
```
- 내 폴더면 무조건 1번 Enter

---

## 기본 사용법

| 동작 | 방법 |
|------|------|
| 메시지 전송 | 타이핑 후 Enter |
| 줄바꿈 (전송 안함) | Shift+Enter |
| 취소 | Esc |
| 종료 | `/exit` 또는 Ctrl+C |

---

## 자주 쓰는 슬래시 명령어

```
/help        - 도움말
/clear       - 대화 초기화
/compact     - 대화 요약 (컨텍스트 절약)
/status      - 현재 상태
/cost        - 사용량 확인
```

---

## 파일 작업

- **읽기/수정**: 그냥 말로 요청 ("이 파일 수정해줘")
- Claude가 직접 파일 편집 가능 (허락 필요할 수 있음)
- 허락 팝업 뜨면 `y` = 허용, `n` = 거부

---

## 권한 모드

| 모드 | 설명 |
|------|------|
| 기본 | 매번 허락 물어봄 |
| `--dangerously-skip-permissions` | 모두 자동 허용 (주의!) |

---

## Windows PowerShell에서 실행

```powershell
claude          # 현재 폴더에서 시작
claude "질문"   # 바로 질문 전달
```

---

## 팁

- 질문은 **한국어**로 해도 됨
- 코드 수정 요청 시 파일명/줄 번호 알려주면 더 정확함
- 긴 대화는 `/compact`로 요약해서 컨텍스트 절약
- 잘못된 작업은 즉시 `Esc`로 취소 가능
