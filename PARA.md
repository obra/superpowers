# PARA Memory System

Shared durable knowledge root: `~/life`

## Structure
- `projects/`
- `areas/`
- `resources/`
- `archives/`

## Rules
- Read `summary.md` first.
- Read `items.json` for deeper detail.
- Store durable facts as atomic entries.
- Supersede outdated facts instead of deleting history.
- Never store secrets.
