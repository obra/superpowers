# Glossary

- **Agent**: the coding assistant working in the repo.
- **Bootstrap**: the initial load path that activates repo instructions before task work starts.
- **Domain doc**: a short reference that defines repo-specific language and decisions.
- **Harness**: the host environment that runs the agent and loads Superpowers.
- **Issue tracker**: GitHub Issues in this repository.
- **Skill**: a reusable behavior package under `skills/`.

For repo-specific terms and decision context, see [docs/agents/domain.md](agents/domain.md).
