# Domain Notes

These terms are canonical for this repository.

## Terms

- **Superpowers**: the repository and the agent methodology it packages.
- **Skill**: a reusable instruction bundle under `skills/`.
- **Harness**: an environment that can load Superpowers, such as Claude Code, Codex CLI, OpenCode, Cursor, or Gemini CLI.
- **Bootstrap**: the startup path that loads instructions so skills can trigger automatically.
- **Plugin**: the installable distribution unit for a harness.
- **Issue tracker**: GitHub Issues for this repository.
- **ADR**: a short decision record for a repo-wide choice that should not live only in chat.

## Rule Of Thumb

If a term affects issue triage, skill design, or install behavior, define it here before using it elsewhere.
