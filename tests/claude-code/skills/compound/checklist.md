# Compound Compliance Checklist

## Triviality Assessment (COMPULSORY)

- [ ] Triviality assessment performed (problem evaluated for non-triviality)
- [ ] Problem correctly identified as non-trivial (multiple attempts were made)

## Solution Quality Gate (COMPULSORY)

- [ ] Solution Quality Gate appeared in response
- [ ] Symptoms section includes exact error message (quoted): "ReferenceError: userData is not defined"
- [ ] Failed Attempts section populated (at least one entry for the failed import attempt)
- [ ] Root Cause explains WHY the problem occurred (not just "fixed the scope issue")
- [ ] Solution has step-by-step instructions (actionable steps, not vague guidance)
- [ ] Prevention section has actionable items (how to avoid in future)

## Pattern Detection Gate (COMPULSORY)

- [ ] Pattern Detection Gate ran (executed `ls docs/solutions/` or similar command)
- [ ] Pattern check result noted (even if 0 similar issues)

## Solution Document

- [ ] Solution doc saved to `docs/solutions/{category}/` directory
- [ ] Solution doc uses correct category (runtime-errors or logic-errors)
- [ ] Solution doc filename includes date (YYYY-MM-DD format)
- [ ] Completion announcement made ("Solution captured to...")
