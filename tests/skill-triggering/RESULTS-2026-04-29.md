## Re-running

```bash
mkdir -p /tmp/cw && cd /tmp/cw
claude -p "$(cat /path/to/superpowers/tests/skill-triggering/prompts/committing-work.txt)" \
    --plugin-dir /path/to/superpowers \
    --dangerously-skip-permissions \
    --max-turns 3 \
    --output-format stream-json --verbose \
    > out.jsonl 2>&1
grep '"skill":"superpowers:committing-work"' out.jsonl  # should match
```

Same pattern for pushing-to-remote.

## Note on bundled run-test.sh

The bundled `tests/skill-triggering/run-test.sh` uses the `timeout` command which is GNU
coreutils (not on macOS by default). For macOS users, install via `brew install coreutils`
and use `gtimeout`, or run claude directly per the snippet above. Filing this as a known
issue is out of scope for this PR.
