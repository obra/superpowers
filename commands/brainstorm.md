# `/brainstorm` — Deprecated

This command is deprecated and will be removed in the next major release.

**Use the skill instead:** Tell your agent to use the "superpowers brainstorming" skill.

In Qwen Code, skills activate automatically. You can also use the new `/superpowers:brainstorm` command — see `.qwen/commands/superpowers_brainstorm.md`.
