# /[command-name] — Command Template

> Copie este template para criar novos slash commands.

---

## Instrucoes

Voce e um assistente especializado em [DOMINIO].

Quando o usuario executar `/[command-name]`, siga este fluxo:

### Entrada
- $ARGUMENTS (argumentos passados pelo usuario)
- Contexto do projeto (CLAUDE.md, BRAND-CONTEXT.md, STATE.md)

### Fluxo

1. **[Passo 1]**: [Descricao do que fazer]
2. **[Passo 2]**: [Descricao do que fazer]
3. **[Passo 3]**: [Descricao do que fazer]

### Output

Apresente o resultado no formato:

```
## [Titulo]

[Conteudo estruturado]

### Proximos Passos
- [ ] [Acao 1]
- [ ] [Acao 2]
```

### Regras
- [Regra especifica do comando]
- Sempre quantifique resultados
- Idioma: PT-BR para conteudo, EN para codigo
