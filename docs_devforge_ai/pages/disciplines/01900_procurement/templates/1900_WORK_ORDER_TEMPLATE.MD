# Work Order Template

## Document Overview

This Work Order template is used for authorizing and tracking specific work tasks within civil engineering projects. It can be used for internal teams or external subcontractors to define work scope, schedule, resources, and quality requirements.

## Usage Notes

1. Replace all bracketed placeholders with actual values
2. Ensure WO number follows the format: WO-YYYYMMDD-XXXX
3. Obtain all required approvals before work commences
4. Complete the Work Completion & Sign-Off section upon completion
5. Document any deviations from the original plan
6. Attach all relevant supporting documents (drawings, permits, test reports)
7. Retain completed Work Order for project records
