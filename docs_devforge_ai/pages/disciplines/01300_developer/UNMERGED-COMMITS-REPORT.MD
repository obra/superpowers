## How to View Details for Any Branch

```bash
# See all unmerged commits for a specific branch
git log origin/main..origin/<branch-name> --oneline

# See detailed commit messages
git log origin/main..origin/<branch-name>

# See file changes
git diff origin/main...origin/<branch-name> --stat

# See specific files changed
git diff origin/main...origin/<branch-name> -- path/to/file
```

## How to Merge a Branch

```bash
# Switch to main branch
git checkout main

# Pull latest changes
git pull origin main

# Merge the branch (with merge commit)
git merge origin/<branch-name>

# OR rebase the branch (cleaner history)
git rebase origin/<branch-name>

# Push changes
git push origin main
```
