- Shelf Life: 5 years

### 4. DELIVERY LOGISTICS

#### 4.1 Transportation Requirements
**Vehicle Specifications:**
- Clean, dedicated lubricant transport vehicles
- Stainless steel or epoxy-lined tanks
- Product segregation capabilities
- Temperature controlled compartments

**Driver Requirements:**
- Trained in hazardous materials handling
- Valid dangerous goods license
- Company uniform and identification
- Emergency response training

#### 4.2 Documentation Requirements
**Delivery Note Requirements:**
- Purchase order number
- Product codes and descriptions
- Quantities delivered
- Batch numbers and expiry dates
- Certificate of Analysis reference
- Safety Data Sheet reference

**Quality Documentation:**
- Certificate of Analysis for each batch
- Batch traceability records
- Quality control test results
- Shelf life certification

- Accessibility for inspection
- Spill containment provisions

#### 6.2 Handling Equipment
**Required Equipment:**

- Drum handling equipment
- Spill containment materials
- PPE storage facilities

**Safety Requirements:**
- Trained personnel only
- Proper lifting techniques
- Spill prevention measures
- Emergency response equipment
