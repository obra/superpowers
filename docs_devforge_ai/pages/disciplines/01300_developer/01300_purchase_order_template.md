## Usage Notes

1. Replace all bracketed placeholders with actual values
2. Ensure PO number follows the format: PO-YYYYMMDD-XXXX
3. Attach relevant terms and conditions if not referenced in a master agreement
4. Obtain proper authorization before issuing to vendor
5. Retain signed copy for records
