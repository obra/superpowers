# HSSE Contractor Vetting Appendices Guide (02400 Series)

This directory contains comprehensive HSSE contractor vetting reference documents organized as a modular appendix system for the Safety page (02400), aligned with established procurement workflow patterns.

## =ﾋ **Guide Structure Overview**

Following the successful appendices approach from procurement workflows, this guide organizes HSSE contractor evaluation into modular components. Each appendix represents a complete assessment document covering specific aspects of contractor safety qualification and performance evaluation.

### Scoring Mechanism

#### Section-Based Weighted Scoring
**Field Type:** Calculated display with color-coded indicators
**Field Name:** Total HSE Score Calculation
**Narrative:** An automated scoring system that calculates weighted scores across all evaluation sections and provides final recommendations:
- 4.0+ Points: APPROVED - High HSE Performance
- 3.0-3.9 Points: CONDITIONAL - Moderate HSE Performance, Requires Improvements
- Below 3.0 Points: REJECTED - Poor HSE Performance

#### Manual Section Score Override
**Field Type:** Number inputs (0-5 scale)
**Field Name:** Section Score Manual Entry
**Narrative:** Section evaluators can manually enter overall section scores or let the system calculate averages from individual question scores.

#### Scoring Visibility Toggle
**Field Type:** Button control
**Field Name:** Scoring System Toggle
**Narrative:** A toggle control allowing users to show/hide the scoring interface during evaluation, useful for unblinded assessments.
