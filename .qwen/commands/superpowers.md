# /superpowers - Main Entry Point

Superpowers is a complete software development workflow for coding agents, built on top of composable "skills" that make sure your agent uses best practices.

## Core Workflow

1. **`/superpowers:brainstorm`** → Explore ideas, requirements, and design
2. **`/superpowers:plan`** → Create detailed implementation plans
3. **`/superpowers:execute`** → Execute plans with subagent-driven development

## Quick Start

- Starting a new feature? → `/superpowers:brainstorm`
- Have a spec ready? → `/superpowers:plan`
- Have a plan ready? → `/superpowers:execute`
- Want to see all commands? → `/superpowers:help`

## How It Works

Superpowers automatically activates the right skill for the job:
- **Brainstorming** before any creative work
- **Writing Plans** for multi-step implementation planning
- **Executing Plans** with subagent-driven development
- **Systematic Debugging** when things go wrong
- **Test-Driven Development** for quality code

All commands follow YAGNI (You Aren't Gonna Need It), DRY, and TDD principles with frequent commits.
