# Load-On-Demand Reference Index

These files document tool-name equivalents for non-Claude-Code platforms. They are NOT loaded automatically — read only the file matching your harness.

| Platform | File | When to load |
|---|---|---|
| Claude Code | (none — defaults are Claude Code tool names) | never |
| Copilot CLI | `copilot-tools.md` | when running under Copilot CLI |
| Codex | `codex-tools.md` | when running under Codex |
| Gemini CLI | `gemini-tools.md` | auto-loaded via GEMINI.md; manual fallback if missing |

**How to load:** use the `Read` tool on the file path above, scoped to your platform only. Do not load all of them.

If your platform isn't listed, the Claude Code tool names usually map cleanly — try first, consult your harness docs only if a tool is missing.
