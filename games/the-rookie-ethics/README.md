# The Rookie: Ethics on the Beat

An ethical puzzle game inspired by the TV series "The Rookie" where you play as a rookie police officer facing moral dilemmas during your first shift.

## About the Game

You'll face 8 challenging scenarios that test your judgment on:
- **Integrity**: Your commitment to honesty and moral principles
- **Protocol**: How well you follow department procedures
- **Humanity**: Your compassion and understanding for others
- **Trust**: How much your colleagues and community trust you

Each decision affects your scores and determines your final officer ranking.

## Running the Game

### Quick Start (Development)

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npm start
   ```

### Building for macOS

To create a standalone macOS application:

```bash
# Build DMG installer
npm run build:mac

# Or just build the app
npm run build:dmg
```

The built application will be in the `dist/` folder.

### Running Without Electron

You can also open `index.html` directly in any modern web browser to play the game without installing Electron.

## Scenarios

The game includes 8 ethical dilemmas inspired by situations from The Rookie:

1. **The Shoplifter's Dilemma** - Property crime with extenuating circumstances
2. **The Partner's Secret** - Witnessing misconduct by a fellow officer
3. **The Domestic Call** - Responding to domestic violence
4. **The Chase Decision** - Pursuit protocol and public safety
5. **The Witness Protection** - Protecting a witness under threat
6. **The Body Camera Dilemma** - Evidence integrity and self-reporting
7. **The Informant's Request** - Confidential informant ethics
8. **The Final Call** - Justice vs. mercy

## Themes from The Rookie

This game explores themes central to the TV series:
- Balancing protocol with compassion
- The complexities of modern policing
- Ethical gray areas in law enforcement
- The importance of integrity
- Second chances and redemption
- Building trust with the community
