# Security Policy

## Supported Versions

Only the latest stable release builds are supported with security patches.

## Reporting a Vulnerability

If you identify a security vulnerability in the superpowers project (such as arbitrary command injection via prompts, workspace boundary breakouts from subagents, or sensitive credentials leakage via script telemetry), please do **not** open a public issue.

Please report vulnerabilities privately by emailing the maintainers or using GitHub's private vulnerability reporting feature.
